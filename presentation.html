RESULT 2

The Switch Box

Custom hardware bridging spatial control to physical switches

~15s — SECTION BREAK. Watch exploded view animation. "Our second result: custom hardware." Let the model reassemble before advancing.

Switch Box Overview

A modular ESP32-based enclosure that physically actuates existing wall switches via magnetic modules

ESP32 Controller

Wi-Fi enabled microcontroller handles MQTT messaging and module coordination

Magnetic Modules

Snap-on switch actuators with solenoid-driven toggles for different switch types

Modular Design

Supports single, double, and dimmer switch configurations in one enclosure

HARDWARE

Switch Module Types

Toggle Module

Standard on/off switches. Solenoid-driven with position feedback sensor.

Rocker Module

Multi-position rockers. Dual actuator design for up/down control.

Dimmer Module

Rotary dimmer control. Stepper motor with encoder for precise positioning.

ENGINEERING

Key Design Decisions

MQTT over REST

Pub/sub model enables real-time bidirectional communication with lower overhead than HTTP polling

Magnetic Mounting

Non-destructive installation — attaches over existing switches without rewiring or modifications

Modular Architecture

Swap modules to support different switch types without replacing the entire enclosure
